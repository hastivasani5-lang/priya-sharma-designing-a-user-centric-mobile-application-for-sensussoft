# Designing a User-Centric Mobile Application for Sensussoft

**Difficulty:** Mid-level  
**Deadline:** 3 days

## Scenario

Sensussoft is looking to develop a mobile application that allows users to track and manage their daily habits, including exercise, sleep, and nutrition. The application should be user-friendly, intuitive, and provide personalized recommendations to help users achieve their goals. The target audience is health-conscious individuals aged 25-45.

## Requirements

1. Conduct user research to identify the key features and functionalities required for the application
2. Create low-fidelity wireframes of the application's main screens, including the dashboard, habit tracking, and recommendations
3. Design a visually appealing and consistent UI for the application, incorporating Sensussoft's brand guidelines
4. Develop a user flow that illustrates the application's navigation and key interactions

## Deliverables

- A Figma file (share link) or a PDF of your mockups/wireframes

## Evaluation Criteria

- Clarity and effectiveness of the user flow and navigation
- Quality and consistency of the visual design and UI components
- Depth and insight of the user research and its application to the design
- Overall usability and user experience of the proposed application design
